# Feature Maturity Policy

Production:
Core public-beta journey intended for normal learner use.

Beta:
Useful feature still being validated. It must be labelled clearly.

Simulation:
Demonstrates a professional workflow but does not claim a live production integration.

Hidden:
Not shown in default learner navigation.

The default learner navigation should remain focused on Today, Course Map, Textbook, Practice,
Projects, Mastery, Portfolio, AI Tutor, Mistake Notebook, Notifications and Account.
