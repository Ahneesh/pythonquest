# CodeQuest Deep Curriculum Standard

Every full lesson chapter should include:

1. Concise theory
2. Interactive visual or mental model
3. Worked example with line-by-line explanation
4. Predict-before-run question
5. Guided coding or query exercise
6. Independent attempt
7. End-of-lesson recap
8. Real-world relevance

A learner should be able to explain the concept, predict behaviour and apply it independently.
Completing only the guided script is not sufficient evidence of mastery.
