# Activation and Retention Standard

A learner is activated after passing the first exercise.

The onboarding flow:

1. Select a learning goal.
2. Select an academy.
3. Complete the short diagnostic.
4. Receive the first pathway.
5. Start the first lesson.
6. Pass the first exercise.

Lifecycle notifications should support learning continuation and clear commercial reminders.
The 15-Day Starter Pack must never renew automatically.
