# CodeQuest Adaptive Learning Engine

v49 uses a transparent rule-based recommendation engine. It does not claim to diagnose
ability, intelligence or employability.

The engine considers:

- academy progress;
- concept mastery estimates;
- spaced-review due dates;
- prerequisite ordering;
- career-role relevance;
- completed projects, capstones, missions and releases;
- learner intensity and academy preference.

Every recommended action includes a visible explanation. Learners can rebuild the plan,
change intensity or ignore a recommendation.

The current graph is a pedagogical model rather than a psychometrically validated assessment
system. Future production work should validate mastery calculations against real learner
outcomes and protect against unfair or self-reinforcing recommendations.
